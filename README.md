# Askme. Forum for your questions
VK (BMSTU) Web-debelopment course (Autumn 2025)

### Requirements
- Docker
- Bash

Other dependencies will be installed during the installation directly into the Docker container

### How to run

Copy the repo
```bash
git clone https://github.com/bmstu-labs/Askme.git
```

Set installation script to executable
```bash
chmod +x ./run.sh
```

Run the installation script
```bash
./run.sh
```
